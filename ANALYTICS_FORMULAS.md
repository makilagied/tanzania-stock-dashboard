# Analytics Formulas

This file documents all formulas used in the dashboard analytics logic.

## Shared Formulas (Funds and Stocks)

- **Total Return (%)**
  - `((end / start) - 1) * 100`

- **Daily Return**
  - `(current - previous) / previous`

- **Mean (Average)**
  - `sum(values) / count`

- **Sample Standard Deviation**
  - `sqrt( sum((x - mean)^2) / (n - 1) )`

- **Annualized Volatility (%)**
  - `daily_std_dev * sqrt(252) * 100`

- **Annualized Return (%)**
  - `(((end / start) - 1) * 100) / years`

- **Years in Period**
  - `days_span / 365.25`

- **Max Drawdown (%)**
  - `min((value - running_peak) / running_peak) * 100`

- **Best Day (%)**
  - `max(daily_return * 100)`

- **Worst Day (%)**
  - `min(daily_return * 100)`

## Stock Analytics Formulas

- **20-day SMA**
  - Average of the last 20 closing prices.

- **50-day SMA**
  - Average of the last 50 closing prices.

- **Period High**
  - Maximum session `high` in window (falls back to `close` if `high` is missing/invalid).

- **Period Low**
  - Minimum session `low` in window (falls back to `close` if `low` is missing/invalid).

- **Average Volume**
  - Mean of positive volume values in the selected window.

- **Volume vs Average (%)**
  - `((latest_volume / avg_volume) - 1) * 100`

## Fund Analytics Formulas

- **Period High NAV**
  - Maximum `navPerUnit` in the selected window.

- **Period Low NAV**
  - Minimum `navPerUnit` in the selected window.

- **Latest Spread (%)**
  - `((salePricePerUnit - repurchasePricePerUnit) / navPerUnit) * 100`

## Analytics Windows

- `1w`: last 7 days
- `1m`: last 30 days
- `mtd`: month-to-date (from start of current month)
- `qtd`: quarter-to-date (from start of current quarter)
- `ytd`: year-to-date (from start of current year)
- `all`: full history

### Stock Examples (CRDB fallback, 30 sessions)

Sample values used:

- Start close = `387.85`
- End close = `409.45`
- One daily return example = `(392.55 - 387.85) / 387.85 = 0.012118...` (`1.2118%`)
- Mean daily return = `0.00190676`
- Daily std dev = `0.00865961`
- Best day = `1.2706%`
- Worst day = `-1.1254%`
- Period high = `409.45`
- Period low = `370.73`
- Avg volume = `6790.8`
- Latest volume = `9374`
- SMA20 = `385.3685`

Formula examples:

- **Total Return (%)**
  - `((409.45 / 387.85) - 1) * 100 = 5.5692%`

- **Daily Return**
  - `(392.55 - 387.85) / 387.85 = 0.012118` (`1.2118%`)

- **Mean (Average)**
  - `sum(all daily returns) / 29 = 0.00190676`

- **Sample Standard Deviation**
  - `sqrt( sum((r - 0.00190676)^2) / (29 - 1) ) = 0.00865961`

- **Annualized Volatility (%)**
  - `0.00865961 * sqrt(252) * 100 = 13.7467%`

- **Max Drawdown (%)**
  - `min((close - running_peak)/running_peak) * 100 = -8.4142%`

- **Best Day (%)**
  - `max(daily_return * 100) = 1.2706%`

- **Worst Day (%)**
  - `min(daily_return * 100) = -1.1254%`

- **20-day SMA**
  - `sum(last 20 closes) / 20 = 385.3685`

- **50-day SMA**
  - Not available in this 30-session sample (`need >= 50 sessions`)

- **Period High**
  - `max(high or close fallback) = 409.45`

- **Period Low**
  - `min(low or close fallback) = 370.73`

- **Average Volume**
  - `sum(positive volumes) / 30 = 6790.8`

- **Volume vs Average (%)**
  - `((9374 / 6790.8) - 1) * 100 = 38.0397%`

### Fund Examples (FAIDA, 25 rows from 2026-02-13 to 2026-03-19)

Sample values used:

- Start NAV = `141.0142` (2026-02-13)
- End NAV = `142.6578` (2026-03-19)
- One daily return example = `(141.1265 - 141.0142) / 141.0142 = 0.00079637` (`0.0796%`)
- Mean daily return = `0.00048301`
- Daily std dev = `0.00033232`
- Best day = `0.1529%`
- Worst day = `0.0230%`
- Period high NAV = `142.6578`
- Period low NAV = `141.0142`
- Latest sale price = `142.6578`
- Latest repurchase price = `142.6578`
- Years in sample = `24 / 365.25 = 0.06571`

Formula examples:

- **Total Return (%)**
  - `((142.6578 / 141.0142) - 1) * 100 = 1.1656%`

- **Daily Return**
  - `(141.1265 - 141.0142) / 141.0142 = 0.00079637` (`0.0796%`)

- **Mean (Average)**
  - `sum(all daily returns) / 24 = 0.00048301`

- **Sample Standard Deviation**
  - `sqrt( sum((r - 0.00048301)^2) / (24 - 1) ) = 0.00033232`

- **Annualized Volatility (%)**
  - `0.00033232 * sqrt(252) * 100 = 0.5275%`

- **Annualized Return (%)**
  - `(((142.6578 / 141.0142) - 1) * 100) / 0.06571 = 17.7380%`

- **Years in Period**
  - `days_span / 365.25 = 24 / 365.25 = 0.06571`

- **Max Drawdown (%)**
  - `min((nav - running_peak)/running_peak) * 100 = 0.0000%`

- **Best Day (%)**
  - `max(daily_return * 100) = 0.1529%`

- **Worst Day (%)**
  - `min(daily_return * 100) = 0.0230%`

- **Period High NAV**
  - `max(navPerUnit) = 142.6578`

- **Period Low NAV**
  - `min(navPerUnit) = 141.0142`

- **Latest Spread (%)**
  - `((142.6578 - 142.6578) / 142.6578) * 100 = 0.0000%`
